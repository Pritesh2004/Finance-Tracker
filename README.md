# Personal Finance Tracker with Stock Sentiment Analysis

## Overview
The Personal Finance Tracker is a comprehensive platform that allows users to manage their finances effectively while providing insightful data on stock market trends. It integrates real-time stock information, a dashboard for tracking financial data, and a sentiment analysis feature that analyzes news headlines to gauge the market sentiment for specific stocks.

## Features
- **Personal Finance Management**: Track income, expenses, and savings in an intuitive dashboard.
- **Stock Sentiment Analysis**: Analyze news headlines related to selected stocks and determine market sentiment using AI/ML algorithms.
- **Real-time Stock Information**: Provides up-to-date stock prices, trends, and financial news for selected stocks.
- **User Finance Dashboard**: A visual dashboard that displays key financial metrics, including income, expenses, savings, and investment performance.
- **Budgeting & Expense Tracking**: Set and track budgets for various categories, ensuring better financial management.

## Technologies Used
- **Backend**:
  - Java
  - Spring Boot
  - Spring MVC
  - Spring Data JPA
  - Spring Security
  - MySQL
  - AI/ML Integration for Sentiment Analysis (using libraries like TensorFlow or Scikit-learn)
  - Stock API Integration (e.g., Alpha Vantage, Yahoo Finance API)

- **Frontend**:
  - Angular
  - HTML
  - CSS
  - Chart.js for data visualization
  - Bootstrap for responsive design
